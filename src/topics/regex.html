Regular expressions

Regular expressions (Regex)

      
        const regex = /d(b+)d/g;

        const anotherRegex = /ab+c/;

        const aBigOne = /^(\d{1,2}-){2}\d{2}(\d{2})?$/;

Regular expressions are a way of describing patterns in a string of data, which allows you to search for strings that match that pattern.

Testing Regex

      
        const regex = /hola/;

        regex.test('hola');  // true
        regex.test('hola y adios');  // true
        regex.test('un hola y otro hola');  // true
        regex.test('adios');  // false

Creating a Regex

      
        const regexLiteral = /abc/i;

        const regexInstance = new RegExp('abc', 'i');

If you need to create a regex dynamically, using the regex literal syntax won't work, you'll need to use the constructor.

Testing

      
        const regex = /hello/;
        const string = 'hello world';

        regex.test(string);

        regex.exec(string);

The test method returns a boolean, and the exec method returns an array of matched groups.

Testing

      
        const regex = /hello/;
        const string = 'hello world';

        string.match(regex);
        string.search(regex);
        string.replace(regex, 'hi');

Every string has methods in its prototype that can be used with regex patterns.

Regex Characters

      
        /[abcd]/    // Matches any of the characters in brakets
        /[a-d]/     // Matches any of the characters in brakets
        /[^abcd]/   // Matches any character that is not in brakets
        /[^a-d]/   // Matches any character that is not in brakets
        /./        // Matches any character except line breaks
        /\d/        // Matches any digit
        /\D/        // Matches any character except digits
        /\w/        // Matches any alphanumeric character
        /\W/        // Matches any character except alphanumeric characters
        /\s/        // Matches a white space
        /\S/        // Matches any character except white spaces

Boundaries

      
        /^hola/    // ^ Marks the start of the string
        /hola$/    // $ Marks the end of the string
        /\bho/    // \b marks a word boundary

Quantifiers

      
        /ho*/      // Matches the character if it appears 0 or more times
        /ho+/      // Matches the character if it appears 1 or more times
        /ho{n}/    // Matches the character if it appears n times
        /ho{n, }/    // Matches the character if it appears n or more times
        /ho{n, m}/    // Matches the character if it appears between n and m times

Groups

      
        const regex = /(mo)/;

        'amor'.matchAll(regex);
        'mormon'.matchAll(regex);

Flags

      
        'mormon'.match(/mo/g);   // g makes the search global
        'AMOR'.match(/mo/i);    // i makes the search case insensitive
        'monday\nmorning'.match(/^mo/m);    // m allows ^ and $ to match newline characters
        'monday\nmorning'.match(/monday.morning/s);    // s allows . to match newline characters
